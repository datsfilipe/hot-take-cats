# Safe Cats

_Cat's saying their tech takes_

## Visit

[https://safe-cats.vercel.app](https://safe-cats.vercel.app/)

## Run locally

```bash
git clone https://github.com/datsfilipe/safe-cats.git
cd safe-cats
pnpm install
pnpm run dev
```
